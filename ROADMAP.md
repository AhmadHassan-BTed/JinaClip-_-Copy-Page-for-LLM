# Project Roadmap

- [x] Modular Service-Oriented Architecture
- [x] Full Jina Reader API header support

- [x] Right-click context menu and keyboard shortcuts

- [ ] **Firefox Support:** Cross-browser compatibility.
- [ ] **Developer SDK:** Expose an API for other extensions to call JinaClip.
